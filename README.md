# Data_Management

## How to use 
*/api/v1/user: get all user in database

*/api/v1/addUser: add new user

*/api/v1/updateUser/{user-id}: update user

*/api/v1/updateUser/{user-id}: delete user


|  Parameter | Type | Description |
| ------------- | ------------- | ------------- |
|  * |   |  root url  |
| user-id   | String  | ID of user |
